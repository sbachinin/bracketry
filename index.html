EASY-PLAYOFFS

WHAT YOU CAN DO WITH IT?

Create a diagram that represents the "draws" of a knockout tournament: But first you have to convert your data to a proper format

You can also make a lot of visual adjustments to your diagram

And it supports live updates

And it has a special mobile layout but it's up to you to decide if mobile layout should be used on a particular screen

BASIC USAGE

You just provide your data and get this default (desktop) view

Custom navigation buttons

You can replace default "scroll buttons" with any other elements or text

"External" navigation buttons

You may override the default navigation behaviour altogether. Use any elements positioned anywhere on your page and call special methods to switch round. live updates
